# Portofolio Website

Website portofolio sederhana berbasis HTML, CSS, dan JavaScript untuk menampilkan biodata, keahlian, proyek, dan kontak.

## Fitur
- Tampilan modern dan responsif
- Navigasi sederhana
- Bagian tentang saya, proyek, dan kontak
- Interaksi menu mobile dan form kontak

## Struktur File
- `index.html` — struktur halaman utama
- `styles.css` — desain dan tata letak
- `script.js` — interaksi sederhana

## Cara Menjalankan
1. Buka folder proyek.
2. Jalankan server lokal, misalnya:
   ```bash
   python3 -m http.server 8000
   ```
3. Buka browser ke `http://localhost:8000`.

## Catatan
Silakan ganti teks seperti nama, deskripsi, dan proyek sesuai identitas Anda.
